# Libery — Стратегічний Голосовий Помічник

PWA додаток для команди клубу книжних авторів **Libery**.  
Голосове ведення стратегічного документу — реєстрація GmbH та розвиток бізнесу.

---

## Що вміє

- 🎙️ Розпізнає голос **українською** та **російською** (перемикання одним дотиком)
- 🤖 Claude AI розуміє команди і редагує стратегічний документ
- 📄 Дописує нові розділи, редагує, показує вміст
- 📌 Зберігає до 30 версій документу з відновленням
- 📱 Встановлюється на телефон як нативний застосунок

---

## Налаштування (5 хвилин)

### 1. Отримай Claude API ключ

1. Зайди на **https://console.anthropic.com**
2. Зареєструйся (email + пароль)
3. Меню зліва → **API Keys** → **Create Key** → назви "Libery"
4. Скопіюй ключ — він виглядає як `sk-ant-api03-...`

⚠️ Ключ показується **один раз** — одразу збережи його!

### 2. Розгорни на GitHub Pages

```bash
# Створи репозиторій libery-assistant на github.com

git init
git add .
git commit -m "Libery Assistant v1"
git remote add origin https://github.com/ТВІЙ_НІКНЕЙМ/libery-assistant.git
git push -u origin main
```

Потім: **Settings → Pages → Source: main → Save**

Через 2-3 хв буде доступно:
`https://ТВІЙ_НІКНЕЙМ.github.io/libery-assistant/`

### 3. Перший запуск

1. Відкрий посилання в Chrome на телефоні
2. Натисни **⚙** (налаштування)
3. Вклей **API ключ**
4. Збережи

### 4. Встанови як додаток

- **Android Chrome:** ⋮ → «Додати на головний екран»
- **iPhone Safari:** 𝖲hare → «На екран "Додому"»

## Голосові команди

| Команда | Результат |
|---------|-----------|
| «Що є в документі?» | Огляд стратегії |
| «Додай розділ про реєстрацію GmbH» | Дописує секцію |
| «Які наступні кроки?» | AI аналізує і підказує |
| «Збережи версію» | Знімок документу |
